# Command: Stress Test

Use this prompt when you want the assistant to challenge the current GAD draft.

```text
Review this GAD artefact like a skeptical architect, security lead, and auditor.

Task:
Find the weakest points, missing controls, unsupported claims, and proof gaps.

Focus on:
- vague scope boundaries
- hidden high-risk decisions
- missing owners
- weak human review design
- regulation claims without implementation consequence
- missing evidence paths
- bypass or incident paths that are not governed

Output sections:
1. Findings ordered by severity
2. Unsupported or risky claims
3. Missing evidence
4. Questions that must be answered before proceeding
5. Tightened replacement wording where useful

Be direct and specific.
```
