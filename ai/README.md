# AI Assistance

AI is useful in GAD work when it behaves like a rigorous delivery partner.

Use it for:

- structuring ambiguous ideas
- drafting first-pass artefacts
- finding missing controls
- stress-testing weak claims
- translating regulations into operational questions

Do not use it for:

- bluffing legal certainty
- inventing controls that do not exist
- collapsing evidence into vague "audit trail" language
- substituting for stakeholder decisions

## Recommended usage pattern

1. Give the assistant the current artefact file.
2. State the system and purpose clearly.
3. Ask for one narrow task at a time.
4. Require assumptions to be listed separately.
5. Ask for challenge points before accepting the draft.
